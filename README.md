# 简介
基于[Nonebot2](https://github.com/nonebot/nonebot2)的滴答清单插件，可通过本插件创建任务、子任务、查询任务

**已支持v2.0.0-beta.2**
## 依赖
- 适配器: onebot v11
- 插件:
    - [nonebot_plugin_apscheduler](https://pypi.org/project/nonebot-plugin-apscheduler/)
